Liburan Saya

Ditulis oleh: Timothy Reinhard Jeremi Pakpahan - 2382010

Halo, nama saya Timothy Reinhard Jeremi Pakpahan. Selama liburan baru-baru ini, saya menghabiskan sebagian besar waktu saya untuk magang. Saya percaya bahwa mendapatkan pengalaman langsung melalui magang akan sangat berharga untuk karier masa depan saya. Pengalaman ini tidak hanya meningkatkan keterampilan saya, tetapi juga memberi saya wawasan tentang dunia profesional, mempersiapkan saya menghadapi tantangan ke depan.

Setelah menyelesaikan magang, saya menghabiskan sisa waktu di rumah dengan melakukan kegiatan yang saya nikmati. Saya menonton berbagai film dan tenggelam dalam membaca ebook dan komik online, seperti manga dan manhwa. Ada begitu banyak cerita menarik hingga saya sering kali membaca komik berjam-jam tanpa menyadari waktu. Kegiatan ini memberi keseimbangan sempurna dengan magang saya, memungkinkan saya untuk bersantai dan bersenang-senang.

Saya sangat bersyukur kepada para penulis dan pembuat komik ini atas karya mereka yang luar biasa. Cerita-cerita mereka membantu mengisi waktu luang saya, terutama saat merasa bosan atau lelah dengan rutinitas. Kreativitas dan dedikasi mereka membuat liburan saya menjadi lebih menyenangkan.
